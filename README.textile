h3. Information

Additional(non-standard) checks for Checkstyle that are compiled as:
- extension to "maven-checkstyle-plugin":http://maven.apache.org/plugins/maven-checkstyle-plugin/ (how to use: "in general":http://maven.apache.org/plugins/maven-checkstyle-plugin/examples/custom-developed-checkstyle.html, "maven example":https://github.com/sevntu-checkstyle/checkstyle-samples/tree/master/maven-project, "ant example":https://github.com/sevntu-checkstyle/checkstyle-samples/tree/master/ant-project).
- extension to "Checkstyle Eclipse plugin":http://eclipse-cs.sourceforge.net/ how to use: install from EclipseCS "update site":
